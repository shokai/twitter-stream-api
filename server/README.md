Twitter Stream Viewer
=====================

* Ruby 1.8.7+
* Sinatra 1.3+
* em-websocket


Install Dependencies
--------------------

    % gem install bundler
    % bundle install


Config
------

    % cp sample.config.yaml

edit it.


Run
---

    % ruby development.ru

open [http://localhost:8080](http://localhost:8080)


Deploy
------
use Passenger with "config.ru"
